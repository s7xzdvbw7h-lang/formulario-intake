# Formulario de Intake Terapéutico - Daniela Navarro Coach

Formulario profesional de intake para nuevos pacientes. Diseño paso a paso con marca personalizada.

## Deploy en Vercel

1. Subí este repositorio a GitHub
2. Conectalo en Vercel (Import Project → desde GitHub)
3. Framework: **Vite**
4. Variables de entorno opcionales:
   - `NOTIFY_EMAIL` - Email donde recibir notificaciones (default: daninavarrocoach@gmail.com)
   - `EMAIL_API_KEY` - API key de web3forms.com (gratis, para notificación por email)
   - `GOOGLE_SCRIPT_URL` - URL del Google Apps Script (para guardar en Google Sheets)

## Stack
- React + Vite
- Vercel Serverless Functions
- Google Fonts (Playfair Display + DM Sans)
